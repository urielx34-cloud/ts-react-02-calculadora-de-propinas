## Cómo ejecutar el proyecto

Clonar el repositorio:

```bash
git clone https://github.com/urielx34-cloud/ts-react-02-calculadora-de-propinas.git

Entrar al proyecto:

```bash
cd ts-react-02-calculadora-de-propinas
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```
